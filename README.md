# Projet Apple

Objectif : À l'aide de nos connaissances en HTML et CSS, essayer de créer une page à l'identique de la page d'accueil du site internet de la marque Apple.
L'objectif second était d'utiliser le terminal pour push et pull sur différentes branches avec les différents collaborateurs, générer des merge conflits et les résoudre

Statut : en pause, il s'agissait d'un exercice pratique mené en cours

Auteurs : Alizée, Godefroy, Laurent
